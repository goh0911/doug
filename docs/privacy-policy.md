# プライバシーポリシー（Privacy Policy）

最終更新日：2026年3月

---

## 1. 概要

Doug - Comic Book Translator（以下「本拡張機能」）は、ユーザーのプライバシーを尊重します。本ポリシーは、本拡張機能がどのようなデータをどのように扱うかを説明します。

## 2. 保存するデータ

本拡張機能がローカルに保存するデータは以下のみです。開発者のサーバーへのデータ送信は一切ありません。

| データ | 保存場所 | 目的 |
|--------|----------|------|
| AI サービスの API キー | `chrome.storage.local`（端末内のみ） | API 認証 |
| 翻訳先言語・モデル等の設定 | `chrome.storage.local`（端末内のみ） | 設定の保持 |
| 翻訳キャッシュ | `chrome.storage.local`（端末内のみ） | API 呼び出しの削減 |
| ホワイトリスト（有効サイト一覧） | `chrome.storage.sync`（Chrome 同期） | 複数端末での設定共有 |

### chrome.storage.sync について

ホワイトリストの保存に `chrome.storage.sync` を使用しています。このデータは **Google のサーバーを経由してお使いの Chrome アカウントに同期**されます。Google によるデータの取り扱いについては [Google のプライバシーポリシー](https://policies.google.com/privacy) をご参照ください。

## 3. 外部サービスへの送信

翻訳実行時、コミックページの**画像データ**が以下のいずれかに送信されます。送信先はユーザーが設定ページで選択した AI プロバイダーに限られます。

| プロバイダー | 送信先 | 再学習ポリシー |
|-------------|--------|--------------|
| Google Gemini | `generativelanguage.googleapis.com` | [Gemini API 追加利用規約](https://ai.google.dev/terms) を参照 |
| Anthropic Claude | `api.anthropic.com` | [Anthropic プライバシーポリシー](https://www.anthropic.com/legal/privacy) を参照 |
| OpenAI | `api.openai.com` | [OpenAI プライバシーポリシー](https://openai.com/policies/privacy-policy) を参照 |
| Ollama | `localhost`（端末内のみ） | 外部送信なし |

各プロバイダーにおける画像データの取り扱い（学習利用の有無等）は、各社のポリシーに従います。

## 4. 収集しないデータ

- 氏名・メールアドレス等の個人情報
- 閲覧履歴・翻訳履歴（開発者への送信なし）
- 利用統計・クラッシュレポート・アクセス解析

## 5. データの削除方法

ローカルに保存されたすべてのデータは以下の方法で削除できます。

1. **設定ページ**からキャッシュをクリア（翻訳キャッシュのみ）
2. **Chrome の拡張機能管理画面**で本拡張機能をアンインストール（すべてのデータが消去されます）

## 6. 対象年齢

本拡張機能自体に年齢制限はありませんが、利用する AI サービスおよびコミックサービスの年齢制限に従ってください。

## 7. ポリシーの変更

本ポリシーは予告なく変更される場合があります。重要な変更が生じた場合は、拡張機能のアップデートノートまたは配布ページにてお知らせします。

## 8. 準拠法

本ポリシーは日本法に準拠し、日本法に従って解釈されます。
